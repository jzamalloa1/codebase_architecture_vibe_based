Project Overview

This project implements a chat application with a Python backend (Flask) and a React frontend. It uses LangChain to interact with OpenAI's language models, providing a clean, user-friendly interface for having conversations with AI assistants.

Key Components

ChatAgent (chat_agent.py)

Implements the core chat functionality using LangChain and OpenAI language models:

  • Initializes with specified model and temperature
  • Maintains conversation context
  • Handles API calls to OpenAI
  • Implements secure error handling for API keys
  • Returns responses from the language model

Flask API (app.py)

Provides REST endpoints for the frontend to interact with:

  • /api/chat - Process messages and get responses
  • /api/new_session - Create a new chat session
  • /api/select_model - Choose a language model
  • /api/models - Get available models
  • /api/health - Check system status

Configuration (config.py)

Manages application settings and model definitions:

  • Available models and their details
  • Default model and temperature settings
  • Environment variable handling
  • API key security

React Frontend (App.js)

User interface with components for:

  • Model selection with descriptions and parameters
  • Chat interface with message history
  • Temperature adjustment for response creativity
  • Error handling and API key validation
  • Responsive design for various devices
